Terms

이용 약관

다운로드가 발생한 시점에서 아래의 모든 가이드라인을 확인하였다고 간주합니다.

사용 가능 범위

  • UTAU / OpenUtau 및 동일 규격을 지원하는 프로그램에서 사용 가능
  • 직접 창작한 곡의 보컬 또는 일반 커버에 사용 가능
  • 조건 준수 시 다른 캐릭터의 CV로 사용 가능
  • 음원이 포함되지 않은 CV의 상업적 이용 가능

핵심 금지 사항

  • 소유권 주장, 자작 발언, 재배포 금지
  • 저작권 침해 및 비방 목적 사용 금지
  • AI 학습 / 레퍼런스 / 추론 등 직접 이용 금지
  • 정치 및 종교적 목적의 이용 금지

기본사항

"MANU"의 음원에 대한 모든 저작권은 "MARU(@MARU_VOC / HUMARU_TWT)에 있음을 명시합니다.

해당 음원은 UTAU용으로 만들어졌으며, UTAU 및 동일한 규격을 지원하는 다른 프로그램에서 사용이 가능합니다.

위에서 서술한 "동일한 규격"이란, 해당 파일의 구조를 변경하지 않고, 사용가능한 소프트웨어를 의미합니다.

다운로드가 발생한 시점에서, 이하의 모든 가이드라인을 확인하였다고 간주하며, 가이드라인 미준수로 인한 모든 내용을 감수하는것으로 간주합니다.

금지사항

아래 사항을 금지합니다.

  • 해당 음원의 소유권 주장 (자작 발언 등)
  • 해당 음원을 이용한 저작권 침해 행위
  • 해당 음원의 재배포 행위
  • 해당 음원을 직접적으로 AI에 이용하는 모든 행위 (학습 / 레퍼런스 / 추론 등의 모든 행위)
  • 해당 음원을 개인 또는 단체의 비방 목적으로 사용하는 행위
  • "이용에 관해" 부분을 미준수하는 행위
  • 해당 내용에 포함되어있지 않더라도, 저작권자 "MARU"의 판단하에 임의적으로 금지가 가능하며, 사용자는 이를 따라야 합니다.

기타 이용에 관해

공통

  • UTAU / OPENUTAU 또는 파일 규격이나 구조를 변경하지 않고 사용이 가능한 모든 프로그램에서 사용이 가능합니다.
    • 단 해당 프로그램이 AI에 직접적으로 사용되는 구조의 프로그램에서는 허용하지 않습니다.
  • 해당 음원을 다른 캐릭터의 CV(Character Voice)로 사용이 가능합니다. 사용시 아래 조건을 추가로 따릅니다.
    • CV는 영상, 이미지 등에 등장하는 모든 캐릭터를 의미합니다.
    • 영상 초반, 후반에 아래 작성한 설명 내용이 영상 화면의 1/2 이상 차지해야하며, 5초 이상 출력되어야 합니다.
    • 영상의 설명란, 또는 텍스트로 작성 가능한 영역에 아래와 같은 내용을 작성해야 합니다.
    • 해당 내용의 경우, 자신이 직접 창작한 곡의 보컬이나 단순한 곡 커버에 사용된경우에 한하여, 영상에 작성할 필요가 없습니다.
    해당 영상에 사용된 음성은 UTAU 프로그램 'MANU'의 음성을 사용하였으며, 해당 음성은 영상에 사용된 캐릭터와 무관합니다.
  • 해당 음원을 사용하실 경우, AI 표기가 필수인 경우, AI 표시가 필요로 합니다.
    • 단, 플랫폼 규약에 따라 표시할 필요가 없을 경우, 표시하지 않아도 됩니다.

    (해당 음원은 Diffsinger로 제작된 음성을 기반으로 만든 UTAU 기반 음성이기 때문에, 플랫폼에 따라 표시가 필요할 수 있습니다. 양해 부탁드립니다.)

  • 해당 음원 사용 시, 반드시 "UTAU MANU"를 사용하였음을 명시해야 합니다.
  • 약관은 저작권자의 판단으로 임의로 교체될 수 있습니다. 교체시 따로 공표되지 않을 수 있습니다.

상업적 이용

  • 해당 음원을 사용한 CV의 경우, 음원이 포함되지 않았다면 자유로운 사용이 가능합니다.
  • 해당 음원을 사용한 음원의 판매의 경우, 저작권자와의 연락이 필요합니다.
  • 해당 음원을 판매하는 행위는 금지합니다. (위 금지사항 참조)

정치/종교적 이용

  • 해당 음원을 이용한 모든 행위를 금지합니다.
  • 캐롤 등 종교적 목적이 아닌, 다른 커버의 경우, 사용이 가능합니다. 이 내용은 저작권자 "MARU"의 판단에 따릅니다.
